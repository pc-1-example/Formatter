Гайд по форматам

Добро пожаловать!

Это интерактивный гайд, который поможет вам изучить основы популярных форматов разметки и визуализации. Выберите один из форматов ниже или перейдите в общую песочницу, чтобы начать экспериментировать.

Markdown

Простой и популярный язык для форматирования текста.

Mermaid

Создание диаграмм и блок-схем из текста.

LaTeX (MathJax)

Набор макрорасширений для верстки математических формул.

PlantUML

Создание UML-диаграмм из простого текстового описания.

Vega-Lite

Декларативный язык для создания интерактивных визуализаций.

AsciiDoc

Более мощный аналог Markdown для технической документации.
