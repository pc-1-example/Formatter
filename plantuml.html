Минимальный пример PlantUML

Ниже приведен пример диаграммы последовательности, сгенерированной сервером PlantUML.

Код для диаграммы:


@startuml
Alice -> Bob: Authentication Request
Bob --> Alice: Authentication Response
Alice -> Bob: Another message
@enduml

Изображение генерируется путем кодирования этого текста в URL.
