Минимальный пример Mermaid

Ниже приведен пример простой блок-схемы, созданной с помощью Mermaid.

            graph TD;
                A[Начало] --> B{Это ромб?};
                B -- Да --> C[Отлично!];
                B -- Нет --> D[Попробуй еще раз];
                C --> E[Конец];
                D --> E;

Код для диаграммы:


graph TD;
    A[Начало] --> B{Это ромб?};
    B -- Да --> C[Отлично!];
    B -- Нет --> D[Попробуй еще раз];
    C --> E[Конец];
    D --> E;
